Guessing Game

Things this script does:

  1. Instantiates a variable to keep track of the number of correct answers
  2. Instantiates an object for each of the three questions. Each object includes:
    1. The text of the question
    2. The correct answer to the question
    3. The answer submitted by the user
    4. Whether the submitted answer is correct (sorta unnecessary)
    5. The text to present if the question is answered correctly
    6. The text to present if the question was answered incorrectly
  3. The script then proceeds to, for each question via do...while loops
    1. Prompt the user for a response to the question, storing the response to the qN.submittedResponse variable
    2. Validate the user imput to a limited degree (i.e. ensure a yes or no was answered)
    3. If invalid input, ask again
    4. If valid input, determine if the response is corrent and increment the correct answer counter, if appropriate.
  4. Once all three questions have been asked, the user is presented with an alert for each question telling them whether their response was correct or not. The text used is either qN.responseCorrect or qN.responseIncorrect.
  5. Finally, the user is presented with the total number of questions they answered correctly in the form of another alert.
